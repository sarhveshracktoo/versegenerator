My Bible Verse Generator Project 📖✨

A few years ago, it was my dream to create a website where anyone could freely access God's Word, discover Bible verses, and grow closer to Scripture without ads, subscriptions, or distractions.

At that time, I had no knowledge of coding, HTML, JavaScript, APIs, or website development. Building a website seemed impossible. But thanks to modern AI tools such as Claude AI and Cursor, I was finally able to turn that dream into reality and launch my own Bible web application.

**Philippians 4:8** has always been an inspiration for this project:

*"Finally, brethren, whatsoever things are true, whatsoever things are honest, whatsoever things are just, whatsoever things are pure, whatsoever things are lovely, whatsoever things are of good report; if there be any virtue, and if there be any praise, think on these things."*

The goal of this website is simple: make Bible reading, Bible study, and daily encouragement available to everyone, completely free.

## Features

### 📅 Daily Verse

Every day, the website automatically displays a Bible verse selected for that specific date. This provides users with fresh daily encouragement and a new Scripture to meditate on each day.

### 📖 Verse of the Week

In addition to the daily verse, the website includes a Verse of the Week feature. Each week displays a carefully selected verse along with a short note or reflection. The system automatically follows the calendar and updates throughout the year.

### 🎲 Random Verse Generator

Users can instantly generate random Bible verses with a single click. This feature is perfect for daily inspiration, personal devotion, encouragement, or discovering passages they may not have read before.

### 📚 Generate Verses From Specific Books

Instead of generating verses from the entire Bible, users can choose a specific book such as Psalms, Proverbs, John, Romans, Genesis, Isaiah, and many others. This allows focused study while still enjoying random verse generation.

### 📖 Complete Bible Reader

The website includes a full Bible reading mode that allows users to browse books, chapters, and verses directly within the app. Users can navigate through Scripture without needing another website or application.

### 🌍 Up to 80 Bible Translations

Readers can choose from dozens of Bible translations, making the Bible accessible to people from different backgrounds and reading preferences. Switching translations is quick and synchronized across the application.

### 🔍 Fast Book and Chapter Navigation

Books and chapters can be selected quickly, making it easy to jump to any part of the Bible without endless scrolling or searching.

### 🔠 Adjustable Font Size

Users can increase or decrease text size for better readability, making the website more comfortable for both desktop and mobile users.

### 📋 Copy Verse

Any displayed verse can be copied instantly to the clipboard for note-taking, Bible study, sermons, devotionals, or sharing with friends and family.

### 📱 Share to Social Media

Users can quickly share verses through WhatsApp, X (Twitter), and other sharing options available on their device.

### 🔊 Text-to-Speech Audio

The website can read Bible verses aloud using built-in text-to-speech technology. This helps users who prefer listening, have visual difficulties, or want to hear Scripture while multitasking.

### 🎨 Bible Verse Image Generator

Users can generate shareable images that include the displayed Bible verse. These images can be posted on social media, used as wallpapers, shared with friends, or included in church groups and ministries.

### 📲 Install as an App

The website supports installation directly from the browser. Users can add it to their desktop or mobile home screen, allowing it to function like a dedicated app without needing to download it from an app store. On mobile browsers, users can tap the **three dots menu** and choose **Add to Home screen**. On desktop, users can click the **three dots menu** in the top-right corner, go to **Cast, save and share**, then select **Install page as app**. This method may vary depending on the browser, but it works best on Chrome.


### ⚡ Optimized for Speed

The website is designed to load quickly and remain lightweight, even on slower internet connections, providing a smooth experience across different devices.

### 🚫 Completely Free

No subscriptions. No premium plans. No advertisements. No hidden fees. The project was created simply to help people access and engage with God's Word.
